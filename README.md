# Zilog Trainer Board Firmware

## Project Title
Custom Zilog Microcontroller Trainer Board with Input/Output Expansion Modules

---

## Project Objective

The main objective of this project is to design and develop a custom trainer board based on a Zilog microcontroller that provides an easy-to-use platform for learning embedded systems, digital electronics, and microcontroller programming.  

This project aims to:

- Develop a functional microcontroller training board using Zilog architecture.
- Provide access to PORT A, PORT B, and PORT C for external interfacing.
- Implement input and output modules for testing digital signals.
- Integrate UART serial communication for debugging and data transfer.
- Add analog input using a potentiometer for ADC experiments.
- Provide regulated 5V and 3.3V power outputs.
- Create a reusable educational platform for students and hobbyists.

---

## Project Description

The Zilog Trainer Board is a custom-built microcontroller development platform designed for practical learning and prototyping. The board includes dedicated headers for multiple I/O ports, making it easy to connect LEDs, switches, sensors, relays, displays, and other external devices.

The system is divided into three major sections:

### Main Control Board
Contains the Zilog microcontroller, reset circuit, power regulation, serial communication interface, and analog potentiometer input.

### Input Module
Provides switches, buttons, and external signal inputs that can be read by the microcontroller through designated ports.

### Output Module
Contains LEDs and output headers used for displaying logic states, controlling loads, and testing programmed outputs.

The board is ideal for laboratory exercises, student projects, embedded systems training, and rapid prototyping applications.
